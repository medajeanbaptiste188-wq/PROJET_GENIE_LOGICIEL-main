# ========================================
# GUIDE COMPLET D'INSTALLATION DU PROJET
# ========================================

## 1. PRÉREQUIS
- Python 3.8+
- pip

## 2. CRÉER UN ENVIRONNEMENT VIRTUEL

### Windows (PowerShell):
python -m venv venv
.\venv\Scripts\Activate.ps1

### macOS/Linux:
python3 -m venv venv
source venv/bin/activate

## 3. INSTALLER LES DÉPENDANCES
pip install -r requirements.txt

## 4. CRÉER LES MIGRATIONS
python manage.py makemigrations gestion

## 5. APPLIQUER LES MIGRATIONS
python manage.py migrate

## 6. CRÉER UN SUPERUSER (Admin)
python manage.py createsuperuser

Entrez les informations demandées:
- Username: admin
- Email: admin@example.com
- Password: (choisissez un mot de passe fort)

## 7. DÉMARRER LE SERVEUR
python manage.py runserver

Le serveur sera accessible à: http://localhost:8000

## 8. ACCÉDER À L'ADMIN
- URL: http://localhost:8000/admin
- Username: admin
- Password: (votre mot de passe)

## 9. ACCÉDER À L'API REST
- http://localhost:8000/api/livres/
- http://localhost:8000/api/membres/
- http://localhost:8000/api/emprunts/

## STRUCTURE DU PROJET
bibliotheque_django/

│   ├── views.py             # Vues API

## DÉPANNAGE

### Erreur: "No module named 'django'"
→ Vérifier que l'environnement virtuel est activé
→ Réinstaller: pip install -r requirements.txt

### Erreur: "ModuleNotFoundError: No module named 'config'"
→ Vérifier que manage.py est dans le répertoire racine
→ Vérifier que vous êtes dans le bon répertoire

### Erreur lors de makemigrations
→ Vérifier les erreurs de syntaxe dans models.py
→ Vérifier que gestion est dans INSTALLED_APPS

### Port 8000 déjà utilisé
→ python manage.py runserver 8001
